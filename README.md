## 🌟 What is VisionAI?

VisionAI is a full-stack image classification web app:
| Feature | Description |
|---|---|
| 🌐 Streamlit Web App | Upload any image, get classified in seconds |
| 🤖 AntigravityAI | Claude API explains the result in plain English + fun facts |
| 📚 Multi-class Support | Train on 10+ custom categories (not just happy/sad) |
| 🕘 History Sidebar | Last 10 classifications with thumbnails and AI explanations |
| 🎨 Modern UI | Clean dark design, confidence bars, top-3 predictions |

## 🚀 How It Works

```
User uploads image
       ↓
PIL Image → Preprocess (resize 224×224, normalize)
       ↓
TensorFlow MobileNetV2 model → prediction + confidence scores
       ↓
AntigravityAI (Claude API)
  ├── Plain-English explanation (3 sentences)
  ├── Fun fact about the detected object
  └── Confidence score label
       ↓
Streamlit renders results + saves to session history
```

## 🛠️ Setup & Installation

### 1. Clone the repo

```bash
git clone https://github.com/arushkumar-aiml/visionai.git
cd visionai
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Prepare your training data

```
data/
├── train/
│   ├── cat/          ← put cat images here (20+ recommended)
│   ├── dog/          ← put dog images here
│   └── bird/         ← add as many classes as you like
└── test/             ← optional validation images (same subfolder structure)
```

### 4. Train the model

```bash
python train.py
```

This will:
- Auto-detect classes from subfolder names
- Train a MobileNetV2 transfer-learning model for 10 epochs
- Save the best model to `models/visionai_model.h5`
- Save class names to `models/class_names.json`

### 5. Run the app

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## 🔑 AntigravityAI Setup (Claude API)

1. Get your API key at [console.anthropic.com](https://console.anthropic.com)
2. Either paste it in the sidebar when the app is running, OR set it as an environment variable:

```bash
export ANTHROPIC_API_KEY="sk-ant-your-key-here"
streamlit run app.py
```

Without an API key, the app still works — you just get basic results without the AI explanation.
